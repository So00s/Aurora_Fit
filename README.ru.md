# Aurora Fit
### Разработано в Саровский физико-технический институт филиял НИЯУ МИФИ (SarPhTI MEPhI)
---
## Лучшее приложение для фитнеса на ОС Аврора

**Aurora Fit** – это удобное и функциональное приложение, разработанное специально для пользователей операционной системы "Аврора". Оно поможет вам достичь ваших фитнес-целей, будь то улучшение общей физической формы, укрепление определённых групп мышц или релаксация через йогу и растяжку. Приложение позволяет удобно планировать, отслеживать и выполнять тренировки с учётом ваших предпочтений.

---

## Цели проекта

- Разработка современного, удобного и безопасного фитнес-приложения для отечественной мобильной платформы **ОС Аврора**.  
- Повышение физической активности пользователей за счёт удобного планирования и визуализации тренировочного процесса.  
- Демонстрация возможностей кроссплатформенной разработки с использованием **Flutter для Аврора SDK**.  
- Обеспечение автономной работы приложения без зависимости от внешних серверов или интернет-подключения.  

---

## Решаемые проблемы

- Отсутствие качественных и локализованных фитнес-приложений, совместимых с ОС Аврора.  
- Недостаток простых инструментов для ведения личного тренировочного плана и отслеживания прогресса.  
- Необходимость в оффлайн-режиме работы с сохранением данных на устройстве.  
- Отсутствие удобного, интуитивно понятного интерфейса под российские мобильные устройства.

## Запуск

Чтобы собрать приложение используйте `flutter-aurora build aurora --release`,  
перед этим установив [Flutter SDK для Аврора ОС](https://gitlab.com/omprussia/flutter/flutter)

> [!TIP]
> #### Приложение работает только на Аврора ОС

---

## Начало работы

После нажатия кнопки `Продолжить` происходит переход на экран выбора тренировок, где:

- Отображается расписание тренировок;
- Предоставляется опция "добавить" для каждого дня, чтобы выбрать новую тренировку.

При выборе дня открывается экран с меню, состоящим из:

- Списка "Типы тренировок" в виде кнопок с изображениями категорий.

После выбора категории открывается экран с меню, отображающим список тренировок, включая:

- Названия тренировок;
- Время выполнения;
- Количество сжигаемых калорий;
- Кнопки `подробнее` и `выбрать`.

---

## Выбор тренировки

Нажав кнопку `выбрать`, вы попадете в диалоговое окно для задания времени начала тренировки. После нажатия `Принять`, выбранное время и категория тренировки отобразятся в расписании.

---

## План тренировки

Для просмотра подробного плана тренировки нажмите кнопку `подробнее`. В новом окне вы увидите детальное описание упражнений. Дополнительно можно нажать `подробнее` для каждого упражнения из списка, чтобы получить ещё больше информации.

---

## Выполнение упражнений

Для начала выполнения нажмите `Начать тренировку`. В открывшемся окне будут отображены:

- Название упражнения;
- Секундомер;
- Кнопки `Стоп` и `Далее`.

- Кнопка `Стоп` останавливает секундомер.
- Кнопка `Далее` переносит к следующему упражнению. После завершения всех упражнений вы увидите экран с сообщением о завершении тренировки и кнопкой `Продолжить`, которая вернёт вас к расписанию тренировок.

---

## Виды тренировок

Приложение включает следующие виды тренировок:

- Кардио;
- Силовые;
- Укрепление ног;
- Укрепление пресса;
- Укрепление рук;
- Укрепление спины и осанки;
- Растяжка;
- Йога.

## Техническая реализация

Aurora Fit разработано с использованием **Flutter**, совместимого с ОС Аврора, и архитектуры **MVVM**.  
Приложение оптимизировано для мобильных устройств и эффективно использует системные ресурсы:  
- Не имеет неработающих или приводящих к падению элементов интерфейса;  
- Обеспечивает плавную анимацию и отзывчивый UI;  
- Не требует постоянного интернет-соединения;  
- Реализует сохранение данных на устройстве с последующим восстановлением при повторном запуске.  

Ошибки, приводящие к сбоям, исключены тестированием на реальном устройстве и эмуляторе Aurora SDK.

## Структура проекта

Проект состоит из следующих основных модулей:

| Модуль | Описание |
|--------|-----------|
| **lib/classes/** | Содержит вспомогательные классы и пользовательские UI-компоненты (кнопки, визуальные элементы, сервисы хранения данных). |
| **lib/json/** | Хранит статические JSON-данные и предустановленные тренировочные наборы. |
| **lib/models/** | Определяет модели данных: упражнения, категории, тренировки и типы тренировок. |
| **lib/pages/** | Содержит экраны приложения: расписание, выбор тренировок, выполнение упражнений и просмотр результатов. |
| **lib/services/** | Реализует бизнес-логику: работу с данными, уведомлениями и загрузкой тренировок. |
| **lib/main.dart** | Главная точка входа приложения, инициализирует структуру и интерфейс. |

**Цель проекта** – создание удобного оффлайн-фитнес-приложения, совместимого с отечественной мобильной платформой, с фокусом на локальную обработку данных и персональную мотивацию пользователя.

**Решаемая проблема** – отсутствие качественных оффлайн-фитнес-решений под ОС Аврора с современным интерфейсом и системой планирования тренировок.

---


## Совместимость и требования

- Приложение **совместимо с ОС Аврора версии 5.0 и выше**;  
- Используются только плагины, прошедшие валидацию и доступные в **Aurora SDK**;  
- Приложение проходит проверку на эмуляторе и физическом устройстве;  
- Содержимое приложения не нарушает законодательство и не содержит оскорбительного контента.
